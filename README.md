# ResistenciaDeMateriales - Guía Interactiva
Este proyecto consiste en un portal web educativo diseñado para presentar de forma clara e interactiva diversos temas fundamentales de Resistencia de Materiales.

## Contenido

El portal incluye temas como:

- Principio de superposición
- Esfuerzos cortantes
- Círculo de Mohr
- Curvatura y deformación en vigas
- Método de la doble integración
- Método del área del diagrama de momento
- Estabilidad estructural

Cada sección contiene:
- Introducción teórica
- Explicación del tema
- Ejercicios prácticos

## Tecnologías utilizadas

- HTML5
- Tailwind CSS
- JavaScript
